# Ключевые аналитические находки
## 1. ГЛАВНОЕ ОТКРЫТИЕ
### Проблема:
Диаграммы Ганта в явном виде почти не публикуются в открытом доступе.

### Решение:
Все российские строительные сметы содержат поле **"Трудозатраты (чел-ч)"**, из которого можно восстановить график работ.

### Формула:
Длительность работы (дни) = Трудозатраты (чел-ч) ÷ (8 часов × Количество рабочих)

### Пример из данных:
- Работа: "Валка деревьев" — 84.23 чел-ч
- При 2 рабочих: 84.23 ÷ (8 × 2) = **5.26 дней**
- **Результат:** Автоматическое построение диаграммы Ганта

## 3. ВЫЯВЛЕННЫЕ ПРОБЛЕМЫ ДАННЫХ

### 3.1. Нестандартизированность
Каждый муниципалитет использует свои шаблоны документов.

### 3.2. Машинонечитаемость
70% графиков — сканы или PDF-изображения вместо структурированных данных.

### 3.3. Фрагментарность
Редко встречаются полные комплекты документов в одном месте.

## 4. ИНСАЙТЫ ДЛЯ БИЗНЕСА

### 4.1. Уникальное предложение
Система может стать первым продуктом, который:
- Автоматически строит графики из смет
- Сравнивает планы с реальностью через CV
- Работает с российскими строительными данными

### 4.2. Рыночное преимущество
Понимание специфики местных данных помогает достигнуть лучшего качества моделей, что сложнее повторить конкурентам.

### 4.3. Масштабируемость
Методология работы с данными применима ко всей строительной отрасли РФ.
