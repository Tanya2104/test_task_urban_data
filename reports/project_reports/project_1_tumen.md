# АНАЛИЗ ПРОЕКТА: Площадь Борцов Революции, Тюмень

## 1. ОПИСАНИЕ ОБЪЕКТА ЗАКУПКИ
- Технические характеристики материалов (марка бетона, тип покрытия, породы деревьев)
- Эскизные изображения необходимых для работы материалов
- Требования к качеству работ

**Применение в ML/CV:**
- **Computer Vision:** Эскизы можно использовать для обучения модели генерации/верификации проектов
- **NLP:** Текстовые спецификации — для извлечения сущностей (материалы, нормы)
- **Data Mining:** Формализация требований в структурированный вид

## 2. ПРОЕКТ КОНТРАКТА
- Юридические условия
- Этапы финансирования
- Ответственность сторон

**Применение в ML:**
- Модель анализа рисков (соответствие этапов работ и платежей)
- Извлечение временных рамок и условий приемки

## 3. СМЕТНАЯ ДОКУМЕНТАЦИЯ 
- **ЛСР** (Локальные сметные расчёты) — детализация по каждому виду работ
- **ОСР** (Объектные сметные расчёты) — свод по объекту

В смете есть данные, указанные в человеко-часах, из этих данных возможно составить таблицу Ганта  

## 4. КЛЮЧЕВОЙ ВЫВОД 
Этот проект содержит все необходимые данные для создания ML-модели:

1. Техническое задание + эскизы
2. Сметы как структурированное описание работ
3. Возможность предсказывать:
   - Сроки выполнения (из смет)
   - Стоимость работ
   - Необходимые материалы

## 5. ПРОБЛЕМЫ ДАННЫХ:
- Сметы в формате Excel/PDF требуют парсинга
- Нет явной привязки этапов работ ко времени 
- Эскизы — растровые изображения, а не векторные чертежи
