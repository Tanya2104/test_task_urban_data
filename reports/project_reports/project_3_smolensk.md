# АНАЛИЗ ПРОЕКТА: Центральный сквер, Смоленская область

## 1. ХАРАКТЕРИСТИКА ПРОЕКТА
**Тип:** Полномасштабный проект благоустройства
**Уникальность:** Наличие всех ключевых документов в одном проекте
**Эталонность:** Идеальный пример для создания тренировочного датасета ML-моделей

## 2. КОМПЛЕКТ ДОКУМЕНТОВ И ИХ ВЗАИМОСВЯЗЬ

### **2.1. Техзадание с работами и сроками**
- Детализированный перечень работ
- Конкретные сроки выполнения
- Гарантийные обязательства подрядчика

### **2.2. ЛСР с трудозатратами (чел-ч)**
**Содержит:** Стандартные сметные расчёты с метрикой `чел-ч`
**Важность:** На основе этих данных можно составить диаграмму Ганта

### **2.3. ГРАФИК ВЫПОЛНЕНИЯ РАБОТ**
**Тип документа:** календарный план
**Что содержит:**
- Представление последовательности работ
- Критический путь проекта
- Зависимости между этапами

### **2.4. ВОР (Ведомость Объёмов Работ)**
**Назначение:** Детализация физических объёмов по каждому виду работ
**Связь с другими документами:** ВОР (объёмы) + ЛСР (трудозатраты на единицу) = График (сроки)
