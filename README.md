# Датасет проектов благоустройства для ML/CV системы

## Описание
Собраны и проанализированы данные реальных проектов благоустройства.
Включает текстовые, табличные и визуальные данные для обучения ML и CV моделей.

## Структура
- 4 основные категории проектов
- Аналитические отчеты по каждому проекту
- Примеры кода и преобразования данных

## Применение
Для разработки системы контроля строительства с использованием:
- Machine Learning (прогнозирование сроков, анализ требований)
- Computer Vision (сравнение планов с реальностью, детекция объектов)

## Быстрый старт
### Установка зависимостей:
pip install -r requirements.txt
### Запуск демо-анализа:
python demo.py
### Открыть интерактивный анализ:
jupyter notebook notebooks/01_data_analysis.ipynb

## Технологический стек
- Python 3.9+ (анализ данных, ML)
- Pandas/NumPy (обработка данных)
- Matplotlib/Seaborn (визуализация)
- Plotly (интерактивные графики)
- Jupyter (интерактивный анализ)
